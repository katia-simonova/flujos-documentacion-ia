# Prompt para revisar documentación administrativa

Usa este prompt para revisar un texto administrativo y convertirlo en una versión más clara.

```text
Actúa como asistente de documentación clara.

Objetivo:
Revisar el siguiente texto administrativo y convertirlo en una guía práctica para una persona usuaria.

Instrucciones:
- No inventes datos.
- Separa hechos, inferencias y dudas.
- Detecta información faltante.
- Explica los pasos en orden.
- Usa lenguaje claro.
- Marca límites si algo no está confirmado.
- No añadas datos personales.
- Termina con una lista de próximas acciones.

Texto:
[pegar texto aquí]
```
