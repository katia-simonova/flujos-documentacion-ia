# Checklist de verificación humana

Checklist para revisar documentos creados con apoyo de IA.

## Exactitud

- [ ] Los datos importantes fueron verificados.
- [ ] No hay cifras inventadas.
- [ ] No hay fuentes inventadas.
- [ ] Las fechas fueron revisadas.
- [ ] Las afirmaciones inciertas están marcadas como límites.

## Privacidad

- [ ] No hay documentos personales.
- [ ] No hay direcciones privadas.
- [ ] No hay datos médicos.
- [ ] No hay números de expediente.
- [ ] No hay información sensible innecesaria.

## Claridad

- [ ] El objetivo del documento es claro.
- [ ] La estructura es lógica.
- [ ] Hay pasos concretos.
- [ ] El lenguaje es comprensible.
- [ ] Las acciones siguientes están definidas.

## Uso responsable

- [ ] La IA no sustituye la revisión humana.
- [ ] El resultado fue leído antes de publicarse.
- [ ] Se corrigieron posibles errores.
- [ ] Se indican límites cuando falta información.
